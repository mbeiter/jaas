JAAS Library

This JAAS library provides an extendable framework to easily create JAAS modules with auditing and message queue support.

Components and Usage

Common

The actual JAAS modules, plus interfaces and reference implementations for extension points such as audit and message queue integration.

The following extension points are supported:

  • Auditing
  • Message queues
  • Credential validators (i.e. the component that actually validates a provided credential)
  • Authenticators (i.e. the component that performs the actual authentication, for instance, against a data store like a JDBC database)

Contents:

  • JAAS modules:
    • JAAS module for username / password based authentication, with support for white label domains
  • Authenticators:
    • Dummy authenticator for username / password based authentication: This is meant as an example on how to implement an authenticator, and for testing. Validation in this module passes if the provided password matches the provided username.
  • Validators:
    • Plain text password validator: This is meant as an example on how to implement a password validator and for testing - not for production.
  • Auditing:
    • Audit logger: This implementation of the audit interface logs audit events to the console log, and is usually not intended for production.
  • Message Queues:
    • Message logger: This implementation of the message queue interface logs messages to the console log, and is certainly not intended for production.

JDBC Password Authenticator

This authenticator allows to perform username / password based authentication against databases, with support for white label domains. The database connection is made either through a configurable JDBC connection pool, or through a JNDI (pooled) connection.
